# onerothkoperhour.phiffer.org

_Twenty four Mark Rothko paintings, one per hour._

## Developer setup

Requirements:

-   [PHP](https://www.php.net/)
-   [Composer](https://getcomposer.org/)

```
composer update
php download.php
php rename.php
```
